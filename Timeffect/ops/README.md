# Timeffect Ops Workspace

## Files
- `pipeline-board.csv` — master pipeline sorted by stages (LEADS → INTRO → SCOPE → BUILD → PAID).
- `revenue-scoreboard.md` — running tally vs 2026 targets, with weighted pipeline table.

## Usage
1. Append every lead/opportunity as a new row in `pipeline-board.csv` with stage + next action.
2. Mirror qualified deals into the revenue scoreboard table with expected close + probability.
3. Review/update weekly (Monday AM plan, Friday PM retrospective).
